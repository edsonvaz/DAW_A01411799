Cuestionario 1: BD vs BDMS


  • ¿Qué es entonces una base de datos - DB? Colección de información interrelacionada de manera coherente.

  • ¿En qué casos es conveniente usar bases de datos? En todo escenario donde la información se maneje mediante archivos relacionados entre si. Sobre todo si esta es manejada por varios usuarios.

  • ¿Qué es un sistema de gestión de base de datos - DBMS? Son gestionadores de información diseñados para manejar grande cantidad de datos de manera organizada.

  • Enlista y explica las funciones/responsabilidades que tienen los DBMS: Arreglar la inconsistencia de datos y brindarles orden, facilitar el acceso a información, brindar permisos a varias personas de manejar los mismos datos, proteger la información, interacción con el gestor de archivos, generar copias de seguridad y tener un control de concurrencia.

  • Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos. En un proyecto de servicio social en el cual participe un año completo, habria sido sumamente útil. Esto se debe a que, en termino generales, cuidamos diferentes grupos de niños por un año para enseñarles valores en unas oficinas, sin embargo, subestibamos la cantidad de trabajo administrativo nos hubiera tocado hacer por un año semanalmente ya que se nos pedia guardar carateristicas de las sesiones, tomar listas, hacer inventarios de cosas que hayamos utlizado y entre otras cosas. El haber tenido una base de datos habría facilitado el acceso a toda la información relevante pero pues, no se hizo.
