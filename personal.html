Preguntas a responder:

  1. ¿Cuál es la diferencia entre Internet y la World Wide Web? Se le puede denominar a la internet como la red de redes mientras que la WWW es lo que se utiliza para poder acceder a esa misma red.

  2. ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE? El metodo "GET" es usado para pedir información de alguna fuente en especial. "POST" es usado para mandar datos para crear fuentes. "PUT" es para actualizar fuentes existentes en un servidor. "HEAD" pide lo mismo que "GET" pero sin el cuerpo de la respuesta. Para modificaciones parciales a una fuente se usa "PATCH". Y por ultimo "DELETE" es para borrar fuentes especificas.

  3. ¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué? Usariamos GET ya que se esta solicitando información que ya está en algun servidor.

  4. ¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL? GET

  5. Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error? Significa que todo se encuentra el orden dentro del servidor y el usuario.

  6. ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué? No, ya que este error infiere que el servidor ha sido accedido pero el host no tiene los recursos requeridos para entrar correctamente.

  7. ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué? Si, ya que este número de error indica que hay algo malo con el servidor.

  8. ¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados. Esto nos indica que ya no es recomendado el uso de dichos atributos en HTM5 como: applet, isindex y menú. En la mayoria de los casos son desaprobados porque se actualizaron de una manera mas optima.

  9. ¿Cuáles son las diferencias principales entre HTML 4 y HTML5? HTML5 permite elementos svg, cambia atributos como type/name/summary, y en general es muchisimo mas sencillo de usar debido a las simplificaciones que se lograron hacer al haberse usado HTML4 en el pasado.

  10. ¿Qué componentes de estructura y estilo tiene una tabla? Utilizan diferentes elementos que diferencian secciones dentro de la misma tabla como elemtnos en fila, en columna y en los headers de la misma.

  11. ¿Qué tanto soporte HTML5 tiene el navegador que utilizas? Mi navegador, GoogleChrome, tiene 473 de 555 puntos de acuerdo al website para comprobar esta respuesta.

  12. ¿Cuál es el ciclo de vida de los sistemas de información? El empezar con una version rapidamente reemplazable ya que se usara basicamente como base para desarrollar una mejor verison de la misma y asi sucesivamente. ¿Cuál es el ciclo de desarrollo de sistemas de información? El empezar por un esqueleto que defina el orden de todo el programa, luego llenar poco a poco todas las fuentes y al final añadir detalles para que todo se entienda y tenga sentido.

Definiciones

  • URL = Dirección web usada para identificar una fuente en...la web.
  • MIME = Conocido actualmente como mediatype, es un identificador de dos partes para formatos de archivos y contenidos en internet.
